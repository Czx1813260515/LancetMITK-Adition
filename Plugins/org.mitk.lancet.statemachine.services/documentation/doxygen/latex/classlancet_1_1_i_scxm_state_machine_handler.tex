\doxysubsection{Detailed Description}
This is the event processing abstract class of the state machine unit. 

Monitors various states and signal events of the state machine. This class exists to simplify the methods of external users, and is also one of the extension interfaces reserved for external users.

When you try to extend this part, you should inherit this class and override its \mbox{\hyperlink{classlancet_1_1_i_scxm_state_machine_handler_a2b8e9f203a49c6520918130649859a08}{Init()}}, \mbox{\hyperlink{classlancet_1_1_i_scxm_state_machine_handler_ae45ca63cdf2a1be5079604b11a94f454}{Start\+State\+Machine()}}, \mbox{\hyperlink{classlancet_1_1_i_scxm_state_machine_handler_a342f3f3806bf7b2ceaa3641b426f533c}{Stop\+State\+Machine()}} methods. You can do whatever you want with the rest of the extension parts.

\begin{DoxyPrecond}{Precondition}
The external state must be created or imported, otherwise the function is invalid.
\end{DoxyPrecond}
