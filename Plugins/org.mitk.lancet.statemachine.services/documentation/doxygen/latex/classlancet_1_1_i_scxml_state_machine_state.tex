State description object of state machine. 

The state machine is generally presented as follows and managed in a tree structure.
\begin{DoxyItemize}
\item State\+Type\+::\+Admin
\begin{DoxyItemize}
\item State\+Type\+::\+Module
\begin{DoxyItemize}

\end{DoxyItemize}
\item State\+Type\+::\+Module
\begin{DoxyItemize}
\item State\+Type\+::\+Ordinary
\item ...
\end{DoxyItemize}
\item ...
\end{DoxyItemize}
\end{DoxyItemize}

\begin{DoxyWarning}{Warning}
Only one State\+Type\+::\+Admin instance object can exist in a process. Otherwise, the previous instance will be overwritten
