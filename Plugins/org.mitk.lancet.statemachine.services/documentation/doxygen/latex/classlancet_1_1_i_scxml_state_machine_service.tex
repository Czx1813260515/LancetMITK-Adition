\doxysubsection{Detailed Description}
This is the abstract class of state machine service. 

It manages and loads multi-\/dimensional state and state machine, and provides the entry point of state management module to the outside. Derived classes should inherit {\ttfamily QObject} on the basis of abstract classes.

\begin{DoxyPrecond}{Precondition}

\end{DoxyPrecond}
The middleware must be loaded using the or Qt plug-\/in framework.
