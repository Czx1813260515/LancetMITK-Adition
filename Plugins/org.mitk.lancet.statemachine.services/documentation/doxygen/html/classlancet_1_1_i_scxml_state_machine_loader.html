Detailed Description

State Machine Loader, A complete state consists of attributes, SCXML, and events.

This is implemented based on the QtScxml module, which converts the state of the state machine from the configuration file to the property objects of the

Qt state machine

and the lancet state machine.

This class is not responsible for state machine event processing.

See also
lancet::IScxmlStateMachineState, QScxmlStateMachine, lancet::IScxmlStateMachineLoader::GetOutput()
Note
The user shall implement the following interfaces: lancet::IScxmlStateMachineLoader::GetOutput()

Set the input parameter set of the loader.

These parameters include the scxml state machine configuration file of Qt, the font attribute description file (json) customized by lancet, and the basic attributes of the state to be loaded. For these basic attributes, refer to

lancet::#IScxmlStateMachineState

Parameters
filenameQt style files with scxml suffix. Please enter an absolute file address.
propertyFileNameThe json file described by the user-defined status attribute. Please enter the absolute file address.
typeDefinition property type of state.
manageManager Object.
idUnique ID code of the status.
