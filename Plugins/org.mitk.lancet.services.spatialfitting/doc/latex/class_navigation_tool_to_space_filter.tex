This is the conversion filter between spatial fitting data space and space.

The filter transfers the upstream navigation tool data to the target spatial coordinate system according to the input spatial transformation matrix, and outputs the data to the downstream filter.~\newline


Please refer to \href{https://docs.mitk.org/nightly/classmitk_1_1NavigationDataToNavigationDataFilter.html}{\texttt{ mitk\+::\+Navigation\+Data\+To\+Navigation\+Data\+Filter}} for details.

{\bfseries{Class diagram\+:}}~\newline
 

{\bfseries{Attention}}~\newline
 The pipeline work data flow processing is calculated from right to left. Here, it means to process the output first and then input.
