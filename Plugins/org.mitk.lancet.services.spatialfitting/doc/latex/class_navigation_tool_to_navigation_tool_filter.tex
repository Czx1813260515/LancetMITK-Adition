This is a tool to tool spatial data conversion filter.

This filter will transfer the navigation tool input data to the UCS tool coordinate system.

\begin{DoxyWarning}{Warning}
The filter itself will not monitor the validity of any data. If correct output data is required, please ensure the validity of the input data.~\newline

\end{DoxyWarning}
